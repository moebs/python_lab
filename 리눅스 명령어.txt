
꼭 알아야할 리눅스 기본 명령어 10가지

1. cat (catenate) - 파일의 내용을 화면에 출력하거나 파일을 만드는 명령어
cat 은 파일이름을 인자로 받아서 그 내용을 출력할 때 사용합니다.

cat fname : fname 의 내용을 출력합니다.


2. clear
clear 명령은 기존 터미널 화면을 지우는데 편리합니다. 
이전의 일부 Linux 명령이 터미널 화면을 깨져서 출력이 깨진 후 터미널 화면을 지워야 하는 경우가 종종 있습니다.


3. cd (change directory) - 디렉토리를 이동하는 명령어

cd ~ : 어느 곳에든지 홈디렉토리로(/home/my_id) 바로 이동합니다.

cd - : 이동하기 바로전의 디렉토리로 이동합니다.

cd .. : 상위 디렉토리로 이동합니다. ( ./ (현재 디렉토리) ../ (상위 디렉토리)) 

cd /dir_a : 절대경로 dir_a 로 이동할 경우 사용합니다.



4. cp (copy)- 파일을 복사하는 명령어

cp 는 파일을 복사하는 명령어 입니다.

cp file-a file-b : file-a 을 file-b 이라는 이름으로 복사합니다.

cp -R dir-a dir-b : dir-a를 dir-b 하위 디렉토리로  복사합니다.



5. ls (list segments) - 현재 위치의 파일목록을 조회하는 명령어

ls 는 현재 위치의 파일 목록을 조회하는 명령어 입니다.


주요 옵션으로는

ls -l : 파일들의 상세정보를 나타냅니다.

ls -a : 숨어있는 파일들도 표시합니다.


ex) ls -al : 파일들의 상세정보를 나타내며, 숨어 있는 것도 표시합니다.



6. man (manual) 

이 명령(man) 다음에 다른 명령의 이름은 해당 명령의 매뉴얼 또는 설명서 페이지를 나열합니다. 


ex) man ls 


7. mkdir (make directory) - 디렉토리를 생성하는 명령어

mkdir 은 새로운 디렉토리를 만들 때 사용하는 명령어입니다.

mkdir dir_name : dir_name 의 디렉토리를 생성합니다.

mkdir -p dir_name/sub_dir : 존재하지 않는 디렉토리의 하위디렉토리까지 생성합니다.



8. mv(move) - 파일을 이동시키는 명령어
mv 는 파일을 이동하는 명령어 입니다. cp 와 비슷하지만 다른 점은 cp 는 파일을 복사하여 원본 파일이 남아있지만 mv 는 원본 파일이 남지 않는다는 점입니다. 그래서 이름 변경시에도 사용가능합니다.

mv file_name_a file_name_b : fname 의 파일을 mfname 의 이름으로 이동/변경 합니다.


9. nano : text 편집기

10. passwd : 패스워드 변경

11. pwd (Print Work Directory)

현재 작업 중인 디렉토리를 보여줍니다. 

12. rm (remove) - 파일을 제거하는 명령어

13. wget (web-get)


rm 은 파일이나 디렉토리를 삭제할 때 사용하는 명령어 입니다.

rm file_a : file_a 을 삭제합니다.

rm -f file_a : file_a 을 묻지 않고 삭제합니다.

rm -r dir_a : dir_a 을 삭제합니다. (디렉토리는 -r 옵션 없이는 삭제할 수 없습니다)
